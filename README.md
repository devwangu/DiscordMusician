## 💬 คำสั่งของบอทที่ใช้งานได้บน Discord (Bot Commands)

เรียกใช้งานโดยการพิมพ์ Prefix `!` ในห้องแชท

- `!play <ชื่อเพลง / ลิงก์>` - เชิญบอทเข้าห้องเสียงและทำการสั่งเล่นเพลง (หรือเพิ่มลงคิวหากกำลังเล่นอยู่)
- `!skip` - ข้ามเพลงที่กำลังเล่นปัจจุบัน ไปยังเพลงถัดไปในคิว
- `!stop` - ยกเลิกคิวทั้งหมด และสั่งให้บอทออกจากห้องเสียง
- `!queue` - เช็ครายการคิวเพลงที่คุณและคนอื่นๆ สั่งเอาไว้ทั้งหมด

## 👨‍💻 สำหรับนักพัฒนา (Developer Note)

- **ทำไมถึงไม่มี `.exe`?** : เพื่อความเบาและป้องกันระบบแอนตี้ไวรัส (False Positive) ของ Windows การใช้โครงสร้าง `สตริปต์.bat` คู่กับ `venv` ถือว่าเป็นวิธีที่เบา เล็ก เข้าใจง่ายที่สุด และยังช่วยให้สามารถแก้ไข Source code ได้ตลอดเวลาโดยไม่ต้องนั่ง Compile ใหม่
- **ไฟล์ที่ห้ามหลุดสู่ Public:** ระบบจะสร้างไฟล์ `config.json` ให้คุณเอง ซึ่งเป็นไฟล์ที่เก็บ Bot Token ของคุณไว้ **ห้ามเปิดเผยไฟล์นี้ให้ใครเด็ดขาด** (ซึ่งมีการตั้งค่า `.gitignore` เผื่อไว้ให้แล้ว)
